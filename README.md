# Object-Oriented Programming in Java

A collection of Java applications demonstrating the core concepts of **Object-Oriented Programming (OOP)** using Java. This repository contains implementations of **Classes & Objects, Encapsulation, Inheritance, Polymorphism, and Abstraction** through practical management system examples.

## 📖 Overview

This project was developed to practice and demonstrate the fundamental principles of Object-Oriented Programming in Java.

### Implemented OOP Concepts

- ✅ Classes & Objects
- ✅ Constructors
- ✅ Encapsulation
- ✅ Inheritance
- ✅ Polymorphism
- ✅ Abstraction
- ✅ Method Overriding
- ✅ Getter & Setter Methods
- ✅ Conditional Statements (`if-else`)
- ✅ `switch` Statements
- ✅ Loops
- ✅ User Input using `Scanner`

# 🚀 Getting Started

## 1. Clone the Repository

```bash
git clone <repository-url>
```

## 2. Open the Project in NetBeans

1. Launch **Apache NetBeans**.
2. Select **File → Open Project**.
3. Navigate to:

```text
object-oriented-programming-java/OOP-Java/CL_BSCD_35_75
```

4. Click **Open Project**.

NetBeans will automatically recognize the project because it contains the `nbproject` folder.

---

## 3. Run the Programs

Each OOP activity has its own entry point (`Main.java`) except the Basic activity.

Run the desired package:

| Activity      | Main Class                 |
| ------------- | -------------------------- |
| Basic         | `Basic.StudentScholarship` |
| Encapsulation | `se.Main`                  |
| Polymorphism  | `Polymorphism.Main`        |
| Inheritance   | `Inheritance.Main`         |
| Abstraction   | `Abstract.Main`            |

You can run any of these classes independently from NetBeans.
